Flight Simulation & Aircraft Dynamics Framework

Project Overview

This project involved the development of a modular flight-simulation framework for a fixed-wing aircraft using Python and Fortran. The simulator models rigid-body aircraft dynamics, actuator behavior, aerodynamic forces, and real-time visualization.

My Role & Contributions

  • Implemented quaternion-based 6-DOF rigid-body equations of motion.
  • Developed aerodynamic force and moment models including stall behavior.
  • Integrated actuator dynamics and control surface response.
  • Created JSON-driven configuration files for simulation inputs.
  • Connected the simulator to real-time visualization tools using Python.

Outcomes & Results

  • Successfully simulated stable and unstable aircraft dynamic modes.
  • Validated short-period and phugoid responses against analytical expectations.
  • Produced a flexible framework suitable for future control-law development.

Tools & Technologies

  • Fortran (numerical integration, dynamics)
  • Python (post-processing, visualization)
  • JSON configuration files
  • Quaternion kinematics and Runge-Kutta integration
